# 🧵 Kriya Boutique

> A fashion designer web app for a boutique that crafts and sells unique, handcrafted clothing.

---

## ✨ About

**Kriya Boutique** is the official online storefront for a one-of-a-kind fashion designer. The platform lets customers browse and purchase exclusive, handmade clothing pieces — each one uniquely designed by the boutique's founder. Every item in the store reflects a personal creative vision, blending artistry with style.

---

## 🌟 Features

- 🛍️ **Shop Unique Pieces** — Browse a curated collection of original, handcrafted clothing
- 👗 **Designer Showcase** — Highlight the boutique's story and creative identity
- 🛒 **Shopping Cart** — Add items and manage your order seamlessly
- 📦 **Product Listings** — View detailed product pages with images, sizes, and pricing
- 📱 **Responsive Design** — Fully optimized for mobile, tablet, and desktop
- 🔐 **Secure Checkout** — Safe and smooth purchase experience

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm (comes with Node.js)

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/kriya-boutique.git

# Navigate into the project folder
cd kriya-boutique

# Install dependencies
npm install
```

### Running the App

```bash
# Start the development server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

### Build for Production

```bash
npm run build
```
